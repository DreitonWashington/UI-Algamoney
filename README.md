# AlgamoneyUi

UI construida no curso Desenvolvedor Angular e Spring da algaworks, foi desenvolvida em Angular 14 e com a biblioteca PrimeNG para componentes

## Subir o serve

`ng serve` para subir o serve no `http://localhost:4200/`.
